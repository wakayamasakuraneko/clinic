    病院紹介

    ※ 当院は猫の不妊手術専門病院です。一般の診療は行っておりません。

    　まずはご予約をお願いいたします。

    当日ご予約無しでの手術は応対出来ない事が多々ありますので、何卒ご了承下さい。

    　いつ現れるか分からない外猫を捕獲したい場合

    まずは定時の餌付けから始めてみて下さい。

    決まった時間帯のみ餌付けし、他の時間は皿ごと片付ける。

    これを1週間程やると猫は時間を学習して、遭遇率が高まります。

    捕まらなくてキャンセルして頂く分には全く問題ありません。

    逆に捕まったから予約無しでなんとかして欲しい、というご要望には対処しかねる場合が多々ありますのでご了承下さい。

    手術の流れと注意点

    ◎ ご予約と手術の流れ

    朝９時頃にご来院して頂き、夕方３～５時にお迎えの流れになります。

    この様なお時間を取れる日でご予約下さい。

    ◎ 連れて来られる際の入れ物について（※重要）

    連れて来られる際の入れ物はプラスチック製のハードキャリーまたは捕獲器でお越し下さい。

    脱走または熱中症の可能性が高いとこちらが判断した場合、受け入れをお断りする事もあります。

    　参照 キャリーケースについて

    釣り針構造の捕獲器は、捕獲器内で暴れた際に、釣り針が目や内臓に刺さる可能性があるため、なるべく使用は控えて下さい。

    　 捕獲器で来院される際の注意補足 　

    ◎ 食事の制限について

    術前８時間は絶食で、お願いいたします。捕獲のための少量のエサ程度であれば、問題ありません。

    ただし、生後３か月未満の子猫の長時間の絶食は危険ですのでご相談下さい。

    水は早朝までならば与えても大丈夫です。
